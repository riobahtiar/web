## Contoh Kode dengan Syntax Highlighting

### Contoh JavaScript

\`\`\`javascript
// Menghitung angka fibonacci
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10)); // 55
\`\`\`

### Contoh TypeScript

\`\`\`typescript
interface BlogPost {
  title: string;
  description: string;
  created_at: Date;
  tags: string[];
}

const post: BlogPost = {
  title: "Artikel Saya",
  description: "Artikel yang bagus",
  created_at: new Date(),
  tags: ["typescript", "astro"]
};
\`\`\`

## Menggunakan Accordion

<Accordion title="Apa itu MDX?" open>
MDX adalah format yang memungkinkan Anda menulis JSX secara mulus di dokumen Markdown Anda. Anda dapat mengimpor komponen, seperti chart interaktif atau alert, dan menyematkannya dalam konten Anda.
</Accordion>

<Accordion title="Mengapa menggunakan MDX untuk blogging?">
MDX memungkinkan Anda membuat konten yang kaya dan interaktif sambil mempertahankan kesederhanaan Markdown. Sangat cocok untuk blog teknis, dokumentasi, dan tutorial.
</Accordion>
